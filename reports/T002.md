## What was done

- `docker-compose.yml` at repo root: `postgres:16` and `redis:7`, both with
  healthchecks, named volumes (`pgdata`, `redisdata`), and all credentials
  sourced from `.env` (`${POSTGRES_USER}`, `${POSTGRES_PASSWORD}`,
  `${POSTGRES_DB}`, `${REDIS_PASSWORD}`; host ports overridable via
  `${POSTGRES_PORT}`/`${REDIS_PORT}` with defaults).
- Redis runs with `--requirepass` so the cache is never open even locally.
- `.env.example` extended with the six new variables as placeholders
  (`change-me` values — never real credentials).
